# INCAR Templates

Use this reference when the user asks for an INCAR template or when selecting a
built-in helper preset. Templates are starting points, not approval. The submit
review must still show the complete effective INCAR and any changed scientific
parameters.

## Category Convention

Group INCAR tags by purpose:

- `global`: system label, cutoff, precision, functional and symmetry choices.
- `electronic`: spin, MAGMOM, parallel layout, electronic convergence, smearing,
  charge initialization, DOS/projection settings, and grid settings.
- `ionic`: relaxation algorithm, force threshold, ionic steps, and cell/shape
  degrees of freedom.
- `output`: restart and charge/output file retention choices.

## magnetic-vdw-relax

Use this preset for magnetic or weakly magnetic vdW relax tests when the user has
reviewed that the MAGMOM count/order matches the POSCAR site order. It is not a
universal relax default.

Helper usage:

```bash
python -m vwf prepare relax \
  --case-root ./case \
  --incar-preset magnetic-vdw-relax \
  --potcar /path/to/POTCAR \
  --kmesh "8 8 8"
```

Template:

```text
# --- global ---
SYSTEM = magnetic vdW relax generated by vwf
ENCUT = 520
PREC = Accurate
GGA = PE
IVDW = 12
LASPH = .TRUE.
LREAL = .FALSE.
ISYM = 0

# --- electronic ---
ISPIN = 2
MAGMOM = 0.05 -0.05 0.05 -0.05 0.05 -0.05 0.05 -0.05
NCORE = 14
LORBIT = 11
ADDGRID = .TRUE.
ISTART = 0
ICHARG = 2
EDIFF = 1E-6
ISMEAR = 0
SIGMA = 0.05

# --- ionic ---
EDIFFG = -0.01
IBRION = 2
NSW = 100
ISIF = 2

# --- output ---
LWAVE = .FALSE.
LCHARG = .FALSE.
```

Review before submit:

- `MAGMOM` has eight entries in the default template; adjust it for any POSCAR
  with a different site count or magnetic ordering.
- `ISIF = 2` keeps cell shape/volume fixed; use only when that matches the
  intended relaxation.
- `ISYM = 0`, `IVDW = 12`, `ISPIN = 2`, `NCORE = 14`, and `GGA = PE` are
  scientific or resource choices and must appear in the submit review.
